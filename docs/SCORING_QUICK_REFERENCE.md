# 股票评分系统 - 快速参考卡

## 🎯 权重分配

```
综合评分 = 技术面 × 35% + 基本面 × 45% + 情绪面 × 20%
```

| 维度 | 权重 | 主要指标 |
|------|------|---------|
| 📊 技术面 | **35%** | RSI, MACD, MA 趋势 |
| 💰 基本面 | **45%** | P/E, ROE, 负债率, 股息 |
| 📰 情绪面 | **20%** | 新闻热度, 新闻情绪 |

---

## 📊 技术面评分标准

### RSI (相对强弱指数) - 40% 权重
| RSI值 | 评分 | 含义 |
|------|------|------|
| < 30 | **80** | 🟢 超卖 - 强势看涨 |
| 30-40 | 20 | 接近超卖 |
| 40-50 | 35 | 偏弱 |
| 50-60 | 50 | 中性偏强 |
| 60-70 | 35 | 偏强 |
| > 70 | 20 | 🔴 超买 - 看跌 |

### MACD (柱状线) - 30% 权重
| 柱状线 | 评分 | 含义 |
|--------|------|------|
| > 0.5 | **80** | 🟢 强势看涨 |
| 0 ~ 0.5 | 60 | 看涨 |
| -0.5 ~ 0 | 40 | 弱势看涨 |
| < -0.5 | 20 | 🔴 看跌 |

### 移动平均线 - 30% 权重
```
价格 > MA20       → +15 分
MA20 > MA50       → +15 分
MA50 > MA200      → +20 分
MAX: 50 分
```

## 💰 基本面评分标准

### P/E 比率 (估值) - 35% 权重
| P/E值 | 评分 | 含义 |
|------|------|------|
| < 15 | **75** | 🟢 低估 |
| 15-25 | 60 | 合理 |
| 25-40 | 40 | 偏高 |
| > 40 | 20 | 🔴 高估 |

### ROE (净资产收益率) - 25% 权重
| ROE值 | 评分 | 含义 |
|------|------|------|
| > 20% | **80** | 🟢 优秀 |
| 15-20% | 65 | 良好 |
| 10-15% | 50 | 平均 |
| 0-10% | 30 | 较弱 |
| < 0% | 10 | 🔴 亏损 |

### 负债率 (财务健康) - 20% 权重
| 负债/权益 | 评分 | 含义 |
|-----------|------|------|
| < 0.5 | **80** | 🟢 财务健康 |
| 0.5-1.0 | 60 | 适度负债 |
| 1.0-2.0 | 35 | 高负债 |
| > 2.0 | 15 | 🔴 过度负债 |

### 股息率 - 20% 权重
| 股息率 | 评分 | 含义 |
|--------|------|------|
| > 5% | 75 | 🟢 高股息 |
| 3-5% | 65 | 中高股息 |
| 1-3% | 55 | 中等股息 |
| 0-1% | 45 | 低股息 |
| 不派息 | 50 | 中性 |

## 📰 情绪面评分标准

### 新闻热度 - 40% 权重
| 新闻数量 | 评分 | 含义 |
|---------|------|------|
| > 10 篇 | 70 | 🔥 热点股 |
| 5-10 篇 | 55 | 中等关注 |

| 正面比例 | 评分 | 含义 |
|---------|------|------|

| 40-60% | 65 | 偏看涨 |
| 35-40% | 50 | 中性 |
| 负面 40-60% | 35 | 偏看跌 |
| 负面 > 60% | 20 | 🔴 强烈看跌 |

## ⭐ 综合评分与评级

### 评分到评级转换
```
综合评分 (1-10) → 评级 → 建议
```

| 评分 | 评级 | 建议 | 颜色 |
|------|------|------|------|
| 8.0-10.0 | 🟢 强烈买入 | 强烈建议买入 | 绿色 |
| 7.0-8.0 | 🟢 买入 | 建议买入/加仓 | 绿色 |
| 5.5-7.0 | 🟡 持有 | 持有或观望 | 黄色 |
| 4.0-5.5 | 🔴 减持 | 考虑减仓 | 红色 |
| 1.0-4.0 | 🔴 强烈卖出 | 强烈建议卖出 | 红色 |

### 风险等级定义
| 评分范围 | 风险等级 | 特征 |
|---------|---------|------|
| 7.0-10.0 | 🟢 **低风险** | 评分高、基本面良好、技术面强势 |
| 4.5-7.0 | 🟡 **中等风险** | 评分中等、某些维度存在隐忧 |
| 1.0-4.5 | 🔴 **高风险** | 评分低、基本面弱或技术面差 |

## 🎯 交易信号示例

### 看涨信号 🟢
- ✅ 技术面强势看涨 (技术评分 > 70)
- ✅ 基本面良好 (基本面评分 > 70)
- ✅ 市场情绪积极 (情绪评分 > 70)
- 📈 近期上涨超 5%
- 🔊 成交量增加

### 看跌信号 🔴
- ⚠️ 技术面走弱 (技术评分 < 30)
- ⚠️ 基本面较弱 (基本面评分 < 40)
- ⚠️ 市场情绪消极 (情绪评分 < 30)
- 📉 近期下跌超 5%

## 🔗 API 端点

### 获取配置
```
GET /api/scoring/config
→ 返回所有权重、标准、定义 (供前端参考)
```

### 获取评分
```
GET /api/scoring/comprehensive/{symbol}
→ 返回完整评分、分布、信号、时间戳
```

## 📝 注意事项

1. **无 API 调用成本** - 所有计算都是固定函数，不调用 Claude API
2. **实时计算** - 基于最新的市场数据（技术指标、基本面、新闻）
3. **透明度** - 所有权重、标准在前端可见，用户能理解评分原理
4. **动态更新** - 当底层数据变化时，评分会自动更新
5. **多维度平衡** - 综合考虑技术、基本面、情绪，避免单一维度偏差

## 🚀 使用示例

### 快速评估一支股票
1. 打开前端，选择股票 (e.g., AAPL)
2. 点击「⭐ Stock Score」选项卡
3. 查看综合评分和评级
4. 查看三维度分布
5. 查看交易信号和建议
6. 参考权重分配和评分标准

### 理解评分原理
- 权重分配：技术面 35%、基本面 45%、情绪面 20%
- 基本面权重最高，因为基本面是股票的长期价值
- 技术面帮助把握短期机会
- 情绪面反映市场情绪，辅助决策

### 风险管理
- 低风险 (7-10): 可以考虑买入
- 中等风险 (4.5-7): 观望或小额持有
- 高风险 (1-4.5): 避免或考虑卖出
