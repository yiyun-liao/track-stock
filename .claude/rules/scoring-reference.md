# 股票評分系統 - 快速參考卡

## 🎯 權重分配

```
綜合評分 = 技術面 × 35% + 基本面 × 45% + 情緒面 × 20%
```

| 維度 | 權重 | 主要指標 |
|------|------|---------|
| 📊 技術面 | **35%** | RSI, MACD, MA 趨勢 |
| 💰 基本面 | **45%** | P/E, ROE, 負債率, 股息 |
| 📰 情緒面 | **20%** | 新聞熱度, 新聞情緒 |

---

## 📊 技術面評分標準

### RSI (相對強弱指數) - 40% 權重
| RSI值 | 評分 | 含義 |
|------|------|------|
| < 30 | **80** | 🟢 超賣 - 強勢看漲 |
| 30-40 | 20 | 接近超賣 |
| 40-50 | 35 | 偏弱 |
| 50-60 | 50 | 中性偏強 |
| 60-70 | 35 | 偏強 |
| > 70 | 20 | 🔴 超買 - 看跌 |

### MACD (柱狀線) - 30% 權重
| 柱狀線 | 評分 | 含義 |
|--------|------|------|
| > 0.5 | **80** | 🟢 強勢看漲 |
| 0 ~ 0.5 | 60 | 看漲 |
| -0.5 ~ 0 | 40 | 弱勢看漲 |
| < -0.5 | 20 | 🔴 看跌 |

### 移動平均線 - 30% 權重
```
价格 > MA20       → +15 分
MA20 > MA50       → +15 分
MA50 > MA200      → +20 分
MAX: 50 分
```

## ⭐ 綜合評分與評級

### 評分到評級轉換
```
綜合評分 (1-10) → 評級 → 建議
```

| 評分 | 評級 | 建議 | 顏色 |
|------|------|------|------|
| 8.0-10.0 | 🟢 強烈買入 | 強烈建議買入 | 綠色 |
| 7.0-8.0 | 🟢 買入 | 建議買入/加倉 | 綠色 |
| 5.5-7.0 | 🟡 持有 | 持有或觀望 | 黃色 |
| 4.0-5.5 | 🔴 減持 | 考慮減倉 | 紅色 |
| 1.0-4.0 | 🔴 強烈賣出 | 強烈建議賣出 | 紅色 |

### 風險等級定義
| 評分範圍 | 風險等級 | 特徵 |
|---------|---------|------|
| 7.0-10.0 | 🟢 **低風險** | 評分高、基本面良好、技術面強勢 |
| 4.5-7.0 | 🟡 **中等風險** | 評分中等、某些維度存在隱憂 |
| 1.0-4.5 | 🔴 **高風險** | 評分低、基本面弱或技術面差 |

## 🎯 交易信號示例

### 看漲信號 🟢
- ✅ 技術面強勢看漲 (技術評分 > 70)
- ✅ 基本面良好 (基本面評分 > 70)
- ✅ 市場情緒積極 (情緒評分 > 70)
- 📈 近期上漲超 5%
- 🔊 成交量增加

### 看跌信號 🔴
- ⚠️ 技術面走弱 (技術評分 < 30)
- ⚠️ 基本面較弱 (基本面評分 < 40)
- ⚠️ 市場情緒消極 (情緒評分 < 30)
- 📉 近期下跌超 5%

## 📝 注意事項

1. **無 API 調用成本** - 所有計算都是固定函數，不調用 Claude API
2. **實時計算** - 基於最新的市場數據（技術指標、基本面、新聞）
3. **透明度** - 所有權重、標準在前端可見，用戶能理解評分原理
4. **動態更新** - 當底層數據變化時，評分會自動更新
5. **多維度平衡** - 綜合考慮技術、基本面、情緒，避免單一維度偏差

## 🚀 使用示例

### 快速評估一支股票
1. 打開前端，選擇股票 (e.g., AAPL)
2. 點擊「⭐ Stock Score」選項卡
3. 查看綜合評分和評級
4. 查看三維度分佈
5. 查看交易信號和建議
6. 參考權重分配和評分標準

### 理解評分原理
- 權重分配：技術面 35%、基本面 45%、情緒面 20%
- 基本面權重最高，因為基本面是股票的長期價值
- 技術面幫助把握短期機會
- 情緒面反映市場情緒，輔助決策

### 風險管理
- 低風險 (7-10): 可以考慮買入
- 中等風險 (4.5-7): 觀望或小額持有
- 高風險 (1-4.5): 避免或考慮賣出
