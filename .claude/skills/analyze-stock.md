# Skill: Analyze Stock（個股投資分析）

## 概述

根據投資哲學框架對個股進行深度分析，提供綜合評分和投資建議。

## 何時使用

- **選股研究** — 評估是否值得投資某支股票
- **持股檢討** — 定期檢視現有持倉的投資邏輯是否仍成立
- **市場機會** — 在特殊市場情況下重新評估標的

## 調用方式

```bash
/project:analyze-stock [SYMBOL]
```

### 範例

```bash
/project:analyze-stock AAPL
/project:analyze-stock MSFT
```

## 必需資訊

- 股票代碼（SYMBOL）— 自動獲取

## 依賴規則

**必讀：**
- @.claude/rules/investment-philosophy.md — 選股哲學、基本面分析、風險管理

## Agents 參與

- **Analyzer Agent** — 執行分析、生成投資建議
- **Scraper Agent** — 獲取即時數據（股價、財務指標、新聞）

## 數據來源

| 數據類型 | 來源 | 更新頻率 |
|---------|------|---------|
| 股價 & 技術指標 | yfinance / Alpha Vantage | 即時 / 1 小時 |
| 財務數據 | Finnhub | 季度更新 |
| 新聞情緒 | NewsAPI / Guardian | 實時 |
| 市場數據 | 各 API 綜合 | 實時 |

## 常見變化

| 場景 | 分析重點 |
|------|---------|
| 成長股（如 NVDA） | 強調 EPS 成長率、競爭優勢、市場前景 |
| 價值股（如 JNJ） | 強調 P/E、股息、現金流、財務穩定性 |
| 周期股（如 XLE） | 強調景氣指標、供需平衡、盈餘波動 |
| 事件驅動（如 Tesla） | 強調催化劑、政策變化、競爭態勢 |
