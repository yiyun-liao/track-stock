# /project:analyze-stock

對指定股票進行完整的投資分析，依照 `.claude/rules/investment-philosophy.md` 的框架輸出結構化報告。

## 使用方式

```
/project:analyze-stock AAPL
/project:analyze-stock TSLA 詳細
```

## 分析框架（依優先序）

參考 `.claude/rules/investment-philosophy.md` 的 AI 分析工具使用指引：

1. **盈餘趨勢** — EPS 近幾季是否持續成長？加速還是減速？
2. **估值合理性** — 目前 P/E 相對歷史、同業、PEG 如何？
3. **競爭地位** — 是否有護城河？行業排名？
4. **財務健康** — 負債比率、自由現金流
5. **技術面輔助** — RSI、均線多空排列（輔助，非主因）
6. **新聞情緒** — 近期重大消息

## 輸出格式

```
## [SYMBOL] 投資分析報告（YYYY-MM-DD）

### 一句話結論
[買入/持有/觀察/避免] — 理由一句話

### 基本面
- EPS 趨勢：
- P/E：X 倍（歷史均值 Y 倍，同業 Z 倍）
- PEG：
- ROE：
- 自由現金流：

### 競爭地位
- 護城河：[有/無/待確認]
- 說明：

### 技術面（輔助）
- RSI：（超賣/中性/超買）
- 均線：MA20/50/200 多空排列

### 風險提示
- [風險1]
- [風險2]

### 結論
[2-3 句總結，明確說明是投資論點還是市場情緒觀察]
```

## 注意事項

- 明確區分「投資論點（基本面）」vs「市場情緒觀察（技術面/新聞）」
- 不因「股價跌很多」就建議買入，看基本面是否仍健全
- 若資料不足，說明缺少什麼資訊，不要硬給建議
