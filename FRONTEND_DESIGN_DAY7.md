```
│
```

```

├─────────────────────────────┤
```

## 🔧 技術實現方案

### 數據獲取策略

```typescript
// 統一的數據獲取 hook
export function useTechnicalIndicators(symbol: string) {
  // GET /api/indicators/technical/{symbol}
  // 緩存 5 分鐘
}

export function useCompanyFinancials(symbol: string) {
  // GET /api/financials/profile/{symbol}
  // GET /api/financials/income/{symbol}
  // GET /api/financials/balance/{symbol}
  // 緩存 1 小時 (財務數據更新不頻繁)
}

export function useGuardianNews(symbol: string) {
  // GET /api/news/guardian/{symbol}
  // 緩存 30 分鐘
}
```

### 圖表庫選擇

| 圖表類型 | 庫 | 原因 |
|---------|-----|------|
| RSI, MACD 線圖 | Recharts | 簡潔, 互動好 |
| Bollinger Bands | Recharts | 複合圖表支持好 |
| 雷達圖 (評分) | Recharts | 5 維度完美 |
| Sankey (現金流) | Recharts | 流向展示佳 |

### 性能考慮

```
Tab 3 (技術分析):
- 後端快速返回 (API 調用 < 1s)
- 圖表渲染優化 (React.memo)
- 初始加載 2-3 個圖表, 其他延遲加載

Tab 4 (財務數據):
- 數據龐大, 分表展示
- 資料卡片獨立加載
- 數據按需展開

Tab 5 (綜合分析):
- 基於 Tab 3 & 4 的數據彙總
- AI 運算在後端 (Claude)
- 顯示預先計算的評分
```
